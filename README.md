# tuapi
Tuner project UI API

## Build

### Install build dependency packages
```sh
apt install libelementary-dev libcairo2-dev libevas1-engines-x cmake
```

### Build
```sh
git clone https://github.com/webispy/tuapi
cd tuapi
mkdir build
cd build
cmake ..
make
```
